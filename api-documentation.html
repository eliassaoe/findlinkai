LinkFinder AI API

Find company URLs programmatically.

Authentication

Include your API key in the Authorization header.

Endpoint

POST https://linkedfinderapiaccess.hamoureliasse.workers.dev/

Request

{ "company_name": "Microsoft" }

Example

curl -X POST "https://linkedfinderapiaccess.hamoureliasse.workers.dev/" \ -H "Content-Type: application/json" \ -H "Authorization: your_api_key" \ -d '{"company_name": "Microsoft"}'
