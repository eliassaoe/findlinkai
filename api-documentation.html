LinkFinder AI API

Complete API documentation for data enrichment services.

Authentication

Include your API key in the Authorization header.

Endpoint

POST https://linkedfinderapiaccess.hamoureliasse.workers.dev/

Available Enrichment Types

1. Company Name to Website

Find a company's official website from its name.

{ "type": "company_name_to_website", "input_data": "Tesla" }

2. Company Name to Phone

Get company contact phone number.

{ "type": "company_name_to_phone", "input_data": "Tesla" }

3. Company Name to LinkedIn URL

Find company's LinkedIn profile URL.

{ "type": "company_name_to_linkedin_url", "input_data": "Tesla" }

4. LinkedIn Company URL to Information

Extract detailed information from LinkedIn company page.

{ "type": "linkedin_company_to_linkedin_info", "input_data": "https://linkedin.com/company/tesla" }

5. LinkedIn Profile URL to Information

Extract information from individual LinkedIn profiles.

{ "type": "linkedin_profile_to_linkedin_info", "input_data": "https://linkedin.com/in/john-doe" }

Bulk Processing

Process multiple inputs by providing an array:

{ "type": "company_name_to_website", "input_data": ["Tesla", "Apple", "Microsoft"] }

Response Format

Success Response:

{ "success": true, "data": { "website": "https://tesla.com" } }

Bulk Response:

{ "success": true, "data": [ { "input": "Tesla", "result": {"website": "https://tesla.com"} }, { "input": "Apple", "result": {"website": "https://apple.com"} } ] }

Error Response:

{ "success": false, "error": { "code": "INVALID_INPUT", "message": "Input data is required" } }

Example

curl -X POST "https://linkedfinderapiaccess.hamoureliasse.workers.dev/" \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "type": "company_name_to_website", "input_data": "Tesla" }'
