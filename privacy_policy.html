Privacy Policy for LinkedIn Finder Add-on

Last Updated: January 2025

Overview

LinkedIn Finder is a Google Sheets add-on that helps users find LinkedIn URLs for people and companies using the LinkFinder AI API. This privacy policy explains how we handle your data.

Information We Collect

API Key Storage

  • Your LinkFinder AI API key is stored securely in your Google account using Google Apps Script's UserProperties service
  • The API key is only accessible by you and is never shared with third parties
  • You can delete your API key at any time through the Settings menu

Spreadsheet Data

  • The add-on only accesses the current spreadsheet you're working with
  • No spreadsheet data is collected, stored, or transmitted except to LinkFinder AI for processing your requests
  • Data sent to LinkFinder AI includes names and company names as needed for LinkedIn URL lookup

Usage Data

  • We do NOT collect any usage statistics, analytics, or telemetry
  • We do NOT track your activities or behavior
  • We do NOT use cookies or similar tracking technologies

How We Use Information

API Key

  • Your API key is used solely to authenticate your requests to the LinkFinder AI service
  • It is stored locally in your Google account and never leaves Google's infrastructure except when making API calls

Spreadsheet Data

  • Data from your spreadsheet (names, company names) is sent to LinkFinder AI's API only when you explicitly request a LinkedIn URL lookup
  • This data is transmitted securely over HTTPS
  • We do not store or log any of your spreadsheet data

Data Sharing

LinkFinder AI

When you use this add-on, the names and company information you specify are sent to LinkFinder AI to process your request. LinkFinder AI's privacy policy applies to data they receive.

Google

This add-on runs on Google Apps Script infrastructure. Google's privacy policy applies to the execution environment and UserProperties storage.

Third Parties

We do NOT sell, trade, or share your data with any third parties except as described above (LinkFinder AI for API requests).

Data Security

  • All communications use HTTPS encryption
  • API keys are stored using Google's secure UserProperties service
  • We follow Google's security best practices for add-on development
  • No data is stored on external servers

Your Rights

You have the right to:

  • Access: Your API key is always visible in the Settings menu
  • Delete: Remove your API key at any time via Settings
  • Control: Choose which spreadsheets to use the add-on with
  • Uninstall: Remove the add-on completely at any time

Permissions

This add-on requests minimal permissions:

  • Access to current spreadsheet only (spreadsheets.currentonly) - Required to read input data and write LinkedIn URLs
  • External requests (script.external_request) - Required to communicate with LinkFinder AI API

Data Retention

  • API Keys: Stored until you delete them via Settings
  • Spreadsheet Data: Never stored by this add-on
  • Logs: We do not maintain logs of your requests

Children's Privacy

This add-on is not directed at children under 13 years of age. We do not knowingly collect information from children.

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last Updated" date.
