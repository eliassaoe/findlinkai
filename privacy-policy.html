Privacy Policy

LinkedIn Profile Finder

Last Updated: January 10, 2025

Introduction

This Privacy Policy describes how LinkedIn Profile Finder ("the Add-on", "we", "our") collects, uses, and protects information when you use our Google Sheets add-on.

Information We Collect

User-Provided Information

  • API Key: You provide your LinkFinder AI API key, which is stored securely in your Google account's user properties
  • Spreadsheet Data: The add-on accesses names and company information from your active spreadsheet to perform LinkedIn profile searches

Automatically Collected Information

  • Usage Data: We do not collect usage statistics or analytics about how you use the add-on
  • Personal Information: We do not collect or store personal information beyond what is necessary for the add-on to function

How We Use Information

The information we access is used solely to:

  1. Authenticate API requests to LinkFinder AI
  2. Process LinkedIn profile search requests
  3. Return results to your spreadsheet
We do NOT:
  • Store your spreadsheet data on our servers
  • Share your data with third parties (except LinkFinder AI for processing)
  • Use your data for marketing or advertising purposes
  • Track your usage patterns
  • Access spreadsheets you haven't explicitly opened with the add-on

Data Storage and Security

API Key Storage

  • Your LinkFinder AI API key is stored in Google's UserProperties service
  • This storage is private to your Google account
  • The API key is only accessible by the add-on and you
  • We do not have access to view or retrieve your API key

Spreadsheet Data

  • Data from your spreadsheet is processed in real-time
  • No spreadsheet data is permanently stored by the add-on
  • Data is only sent to LinkFinder AI for processing your requests
  • Results are written directly back to your spreadsheet

Third-Party Services

LinkFinder AI

The add-on uses LinkFinder AI's API service to find LinkedIn profile URLs. When you use the add-on:

  • Name and company data is sent to LinkFinder AI
  • LinkFinder AI processes this data to find matching LinkedIn profiles
  • LinkFinder AI's privacy policy applies to their handling of this data
  • Please review LinkFinder AI's privacy policy at: linkfinderai.com

Google Services

The add-on operates within Google's infrastructure:

  • Uses Google Apps Script platform
  • Accesses Google Sheets through authorized OAuth scopes
  • Follows Google's security and privacy standards

Data Retention

Data Type Retention Period
API Key Stored until you manually remove it or uninstall the add-on
Spreadsheet Data Not retained; processed only during active use
Processing Data Temporary data used during API calls is not retained after request completion

Your Rights and Choices

You have the right to:

  • Access: Review what data the add-on has access to (your API key and active spreadsheet)
  • Delete: Remove your API key by uninstalling the add-on or through Settings
  • Revoke Access: Remove the add-on's permissions through Google Account settings
  • Data Portability: Your data remains in your spreadsheet under your control

How to Remove Your Data

  1. Open the add-on Settings
  2. Clear your API key (or leave blank)
  3. Uninstall the add-on from your Google Sheets
  4. Revoke authorization from Google Account permissions page

OAuth Scopes and Permissions

The add-on requests the following permissions:

OAuth Scope Purpose
spreadsheets.currentonly Read and modify only the spreadsheet where the add-on is actively used
script.external_request Make API requests to LinkFinder AI service
script.container.ui Display user interface (sidebar and dialogs)

We request only the minimum permissions necessary for the add-on to function.
